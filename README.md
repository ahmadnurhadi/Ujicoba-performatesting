# API Performance Testing with K6

## 🚀 Overview
Project ini bertujuan untuk melakukan benchmarking performa pada sistem menggunakan K6. 
Pengujian mencakup Load Testing dan Stress Testing untuk memastikan stabilitas sistem.

## 🛠 Tools Used
- **K6**: Main performance testing tool.
- **JavaScript**: Scripting language.
- **test.k6.io**: Target environment.

## 📊 Scenarios
1. **Load Test**: Simulating 50 concurrent users to check baseline performance.
2. **Stress Test**: Ramping up to 400 users to find the breaking point.
